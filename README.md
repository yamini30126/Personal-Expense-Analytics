# Personal Expense Analytics

## Overview
This project analyzes personal finance transaction data using Python to identify income vs expense patterns, category-wise spending, and monthly expense trends.

## Technologies Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Google Colab

## Dataset
- **Name:** Personal Finance Data
- **Source:** Kaggle (ramyapintchy/personal-finance-data)
- **File:** Personal_Finance_Dataset.csv
- **Records:** 1,500 rows | 5 columns
- **Time Period:** 2020 - 2024

## Project Description
The dataset contains synthetic personal finance transaction records including date, transaction description, category, amount, and type (Income/Expense). The notebook performs data cleaning, analysis, and visualization to identify spending patterns and financial insights.

## Visualizations
1. Total Income vs Expense - Bar chart comparing overall income and expense
2. Category wise Expense - Bar chart showing spending across categories (Travel, Rent, Food & Drink, etc.)
3. Monthly Expense Trend - Line graph showing expense pattern across 12 months
4. Expense Distribution Pie Chart - Percentage breakdown of expenses by category

## Key Findings
| Finding | Result |
|---|---|
| Total Transactions | 1,500 |
| Total Income | $734,087.00 |
| Total Expense | $1,227,194.37 |
| Net Savings | -$493,107.37 |
| Highest Expense Category | Travel |
| Highest Expense Month | July |
| Lowest Expense Month | December |

## How to Run
1. Open the notebook in Google Colab
2. Upload the Personal_Finance_Dataset.csv dataset
3. Install required libraries
4. Run all cells to reproduce the analysis
