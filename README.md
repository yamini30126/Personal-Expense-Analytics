Personal Expense Analytics

Overview

This project analyzes personal finance transaction data using Python to identify income vs expense patterns, category-wise spending, and monthly expense trends.

Technologies Used


Python
Pandas
Matplotlib
Seaborn
Google Colab


Dataset


Name: Personal Finance Data
Source: Kaggle (ramyapintchy/personal-finance-data)
File: Personal_Finance_Dataset.csv
Records: 1,500 rows | 5 columns
Time Period: 2020 - 2024


Project Description

The dataset contains synthetic personal finance transaction records including date, transaction description, category, amount, and type (Income/Expense). The notebook performs data cleaning, analysis, and visualization to identify spending patterns and financial insights.

Visualizations


Total Income vs Expense - Bar chart comparing overall income and expense
Category wise Expense - Bar chart showing spending across categories (Travel, Rent, Food & Drink, etc.)
Monthly Expense Trend - Line graph showing expense pattern across 12 months
Expense Distribution Pie Chart - Percentage breakdown of expenses by category
